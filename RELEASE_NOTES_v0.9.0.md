# Release Notes v0.9.0

リリース日: 2026-02-26

## ハイライト
- クリップボード履歴アプリとしての主要機能を一通り実装
- タスクトレイ常駐とホットキー起動に対応
- アプリ専用アイコンを追加（実行ファイル・トレイ・各ウィンドウ）
- 履歴ウィンドウのリサイズに対応

## 追加・改善
- クリップボード履歴
  - 履歴の収集・表示・永続化
  - 保存アイテム（タイトル/本文）管理
  - グループ管理
- ホットキー
  - Shift ダブルタップ
  - Ctrl ダブルタップ
  - 組み合わせキー（例: Ctrl/Shift + 任意キー）
- UI/UX
  - 履歴ウィンドウ、設定ウィンドウ、保存/編集ダイアログ
  - タスクトレイメニュー（履歴を開く/設定/終了）
  - 履歴ウィンドウはリサイズ可能
- ログ
  - ホットキーロガーに enable/disable 機能を追加
  - 既定値は無効

## アイコン対応
- `assets/app-icon.ico` を Windows 実行ファイルへ埋め込み
- `assets/tray-icon.rgba` をトレイアイコンとして適用
- ウィンドウ表示時にアプリアイコンを反映

## 永続化ファイル
- `%LOCALAPPDATA%/clip_keeper/clipboard_history.json`
- `%LOCALAPPDATA%/clip_keeper/settings.json`

## 既知事項
- 一部環境では初回起動時にフォーカス制御が OS 側挙動の影響を受ける場合があります
- グローバルホットキーの挙動は他アプリのフック状況に依存します

## 配布物
- Windows 実行ファイル: `target/release/ClipKeeper.exe`
