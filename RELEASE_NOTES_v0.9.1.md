# Release Notes v0.9.1

リリース日: 2026-02-26

## ハイライト
- 非アクティブ化後に Shift ダブルタップで履歴ウィンドウが表示されない問題を修正

## 修正内容
- ホットキーのダブルタップ判定を `KeyPress` 基準から `KeyRelease` 基準へ変更
- 非アクティブ遷移後の押下状態ずれで再表示できなくなるケースを解消
- キーリピート時の誤検知を抑制

## 影響範囲
- ホットキーモード
  - Shift 2回押し
  - Ctrl 2回押し

## 配布物
- Windows 実行ファイル: `target/release/ClipKeeper.exe`
